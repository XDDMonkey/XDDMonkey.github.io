2017-3-10面蚂蚁金服C++实习生

发表于 2017-03-10 |

　　投出去了很多简历，终于今天迎来了第一个面试电话，也是这段时间以来第一次去实验室结果就打来了，虽然问了面试官，面试官明确表示没有二面了，自己还是很开心，毕竟明确了自己的短板，后续还是有机会的。

摘录

  • 自己的项目中遇到的困难，怎么解决的
    　　自己拿websocket说的，比较了轮询和comet技术。
  • 看书方面
    　　问我看了C++方面的什么书，答：《C++ Primer》、《effective C++》、《STL源码剖析》。
  • 设计模式
    　　有没有用static实现过单例模式，如果有，详细叙述一下。
  • C++ 11新特性
    　　用到了C++11中的哪些特性，答：auto，值引用，二维vector。
  • 多态
    　　多态是怎么实现的，问为什么可以用一个指针同时指向父类和子类的对象，引用是否可以起到多态的作用（其实可以，但是自己答了不可以，然后就gg了）。
  • 用C++写过什么
    　　ACM算法，聊天程序什么的，这方面经历比较少。
  • STL
    　　STL里面熟悉哪些，问了一下map的搜索原理和vector的增长原理（问了一下size和capacity之间大小比例关系），按照STL六大组件来说。
  • linux和分布式
    　　还问了一下对gdb是否熟悉，重灾区。

    总结

　　自己主要的短板还是在linux和gdb上面，然后多用C++做一点自己的项目，最后这次面试尚且没有很详细问设计模式，如果详细问了，必然也挂得很严重，这方面自己还要加强，着重看一下C++中stl的实现或者java集合类中运用到的设计模式，并熟悉一下RAII，这次还没有面算法，估计是HR忘了。
