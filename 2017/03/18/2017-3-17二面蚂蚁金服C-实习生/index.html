2017-3-17二面蚂蚁金服C++实习生

发表于 2017-03-18 |

　　之前面试官说我没有二面的，但是昨天早上居然又接到了面试电话，通知我晚上七点半电话面试，窃喜啊。不过这次面试没有问很多理论基础知识，自己把上次面试的时候问到的一些知识都补齐了，结果面试官没有问，上来问我的问题总结一下：

摘录

  • 求交集问题
    　　如果有两个数组，一个有一千万数据，一个有一百万数据，自己首先想到的是用merge-sort和红黑树，然后面试官问我红黑树用哪个建表，哪个来查找。我说一千万的用来建表，这里自己其实是在胡扯，所以还是要学习，另外应该还是有哈希的方法，无奈自己这方面知识不到位。
  • 并发数据访问问题
    　　问了我关于数据库数据访问加锁的问题，一脸懵逼。
  • 问了我写多线程之间同步数据，共享数据的一些问题
    　　自己多线程的经验还是不足，这方面没怎么答。
  • 线程，进程以及数据库的一点基础问题
    　　这方面自己答得还是可以，面试官说我理论知识挺不错，但是实践经验严重缺乏。是的，自己还要多做一些项目来多加练习。

    总结

　　这次面试的时间比较短，半个小时就结束了，同时也比一面暴露出来了更多的问题，自己还要加把劲，不仅要多看书，还要多写代码。
　　结果也问了一下面试官还有没有继续面试的机会，面试官说还要review一下，也没有否决，同时也表示如果还有面试，也会是技术面试。路漫漫其修远兮，如果能顺利，自己至少还有一次技术面试和hr面试，希望自己还能有机会，也希望自己能好好表现，拿到offer。
